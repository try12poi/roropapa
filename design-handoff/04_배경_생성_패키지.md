# 배경 생성 패키지 — 《새벽 3시의 편의점》

> **사용법**: Gemini(또는 ImageFX)에서 **새 채팅**을 열고, 이 파일의 [브리프]를 먼저 붙여넣은 뒤
> BG-1부터 순서대로 생성한다. **캐릭터 대화방과 절대 섞지 말 것** — 이 파일에는
> 의도적으로 캐릭터 정보가 하나도 없다 (배경에 사람이 끼어드는 오염 방지).

---

## [브리프] 새 채팅에 가장 먼저 붙여넣기

```
You are generating background art for a Korean mobile game. Remember these
rules for ALL images in this conversation:

WORLD: At 3 AM, a Korean neighborhood flips into a haunted "other side".
A 24-hour convenience store named "안심24" is the only light in the darkness.
The mood is spooky but cozy — like a ghost story told over warm fish-cake soup.
Never actually scary.

COLOR RULES (fixed hex):
- Night world base: deep navy #0A0E17
- Fluorescent store light: mint-cyan #7FE3F0
- Warm interior glow: cream #FFE9B8
- Item/accent: amber #FFC24B
- Danger accents: small glowing red #FF5A5A eyes hiding in darkness only

HARD RULES:
- NO people, NO characters in any background image
- 2D game background art, cel shading, clean line-art, no photorealism
- Korean convenience store details (parasol table, banana milk, 2+1 posters),
  never Japanese ones
```

## BG-1. 편의점 "안심24" 외관 — 거점 겸 키 비주얼 (P0, 가장 먼저)

```
Wide 16:9 2D game background: a Korean 24-hour convenience store named "안심24"

Atmospheric, cozy yet slightly eerie.

STYLE LOCK: 2D game background art, cel shading with clean line-art, Korean
urban-legend horror-comedy mood — spooky but cozy, never scary, night palette
of deep navy #0A0E17, fluorescent mint-cyan #7FE3F0 lights, warm cream #FFE9B8
interior glow and amber #FFC24B accents, NO people or characters in the scene,
no photorealism, high quality
```

**검수 기준**: 편의점이 "어둠 속 유일한 등대"로 보이는가 / 인도에 빛 웅덩이가 생겼는가

## 재생성 요청 문구 (결과가 어긋났을 때)

- 사람이 나옴 → `Remove all people and characters. The scene must be completely empty of humans.`
- 탑뷰가 너무 밝음/화려함 → `Much darker and flatter. This is a game floor — reduce contrast and remove highlights.`
- 일본풍으로 나옴 → `This must be a KOREAN street/store: Korean signage (한글), parasol table, banana milk.`
- 낮처럼 나옴 → `It must be 3 AM: pitch-dark navy sky, fog, the store light is the only light source.`
- 무섭게 나옴 → `Too scary. Cozier and more playful — warm light, soft shapes, cute hidden red eyes only.`
