# 《새벽 3시의 편의점》 디자인 핸드오프 패키지

Gemini(또는 다른 이미지 생성 AI)에게 넘겨서 게임 아트를 뽑기 위한 자료입니다.

## 파일 구성

| 파일 | 내용 | 용도 |
|---|---|---|
| `01_아트디렉션_브리프.md` | 세계관·톤·색·금지사항 | Gemini에게 **가장 먼저** 통째로 붙여넣기 |
| `02_에셋_리스트.md` | 필요한 에셋 전체 목록 + 규격 + 우선순위 | 작업 순서 관리용 체크리스트 |
| `03_이미지_생성_프롬프트.md` | 에셋별 복붙용 프롬프트 (영문) | 하나씩 복사해서 생성 |
| `04_배경_생성_패키지.md` | 배경 전용 독립 패키지 (브리프+프롬프트+검수기준) | **배경은 반드시 이 파일만으로 새 채팅에서** |
| `assets/` | 확정된 아트 원본 백업 | 캐릭터 시트·턴어라운드 보관소 |

## 사용 방법 (권장 순서)

1. **새 대화를 열고** `01_아트디렉션_브리프.md` 전체를 붙여넣는다.
   - "이 브리프를 기억해. 앞으로 생성하는 모든 이미지는 이 규칙을 따라야 해."라고 지시.
2. `03_이미지_생성_프롬프트.md`에서 **P0(최우선) 에셋부터** 프롬프트를 하나씩 복사해 생성한다.
3. 첫 캐릭터(준호)가 마음에 들면, **그 이미지를 대화에 계속 유지한 채** 나머지 캐릭터를 생성한다.
   - "방금 이 캐릭터와 같은 스타일로"라고 매번 언급 → 스타일 통일의 핵심.
4. 결과물이 어긋나면 브리프의 **금지사항(NEVER)** 목록을 다시 붙여넣고 재생성.

## 대화방 오염 방지 규칙 (실전에서 확인됨)

- 한 대화방에서 여러 캐릭터를 만들면 소품·의상·체형이 서로 섞인다 (스프레이 병 떠돌기, 캐릭터 융합 사례 있음).
- **비교시트(신규 디자인)**: 이어서 해도 됨. 단, 잡소품 오염은 무시.
- **턴어라운드(확정 작업)**: 반드시 **새 채팅** + 해당 캐릭터 레퍼런스 이미지 **1장만** 첨부.
- 프롬프트에 "Do NOT include ..." 금지 목록을 구체적으로 적어도 오염된 방에서는 잘 안 먹힌다 — 새 채팅이 유일한 해결책.

## 스타일이 흔들릴 때 고정하는 팁

- 매 프롬프트 끝의 공통 스타일 블록(`STYLE LOCK`)을 절대 빼지 말 것.
- 캐릭터 시리즈는 한 대화에서 연속 생성 (대화를 바꾸면 스타일이 리셋됨).
- 가장 잘 나온 이미지 1장을 "스타일 기준"으로 지정하고, 이후 생성마다 참조 이미지로 첨부.

## 검수 기준 (통과/반려)

- [ ] 2.5등신인가? (머리 1 : 몸 1.5)
- [ ] 검은 실루엣으로 바꿔도 누구인지 구분되는가?
- [ ] 무서운가? → 반려. "오싹한데 귀여움"이 상한선
- [ ] 적/괴담이 컬러풀한가? → 반려. 적은 무채색+빨간 눈만
- [ ] 배경 없는 캐릭터 에셋에 배경이 깔려 나왔나? → 흰 배경으로 재생성
