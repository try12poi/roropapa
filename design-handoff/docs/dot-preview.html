인게임 도트화 미리보기 — 5인

확정 치비 아트를 실제 인게임 해상도로 자동 변환한 검증용 미리보기. 좌: 원본 / 중: 48px 도트(권장) / 우: 32px 도트
라이더 류헬멧·배달통 실루엣 유지. 단, 전체가 어두워 바이저 발광을 더 밝혀야 함
미화 김주임양호 — 핑크 장갑이 도트에서도 선명. 조끼 줄무늬 살아있음
갸루씨양호 — 핑크블론드+스커트 식별 확실. 리본 망치는 이펙트로 보완 권장
오타쿠 덕규양호 — 빙글빙글 안경이 도트에서도 읽힘. 통통 실루엣 유지
스트리머 지수양호 — 링라이트 2개가 오히려 예쁘게 뭉개짐(수정판에서 1개로)
결론: 48px 기준으로 5인 전원 실루엣·키 컬러가 살아남는다 → 도트 아트 방향 채택 가능. 32px는 안경·키링 등 디테일이 뭉개져 비권장. 이 미리보기는 자동 변환이라 최종 품질이 아니며, 실제 스프라이트는 치비 4방향(정면/뒤/옆) 기준 도트로 다시 제작한다 — 그 전까지는 이 자동 변환본을 프로토타입 플레이스홀더로 사용.
